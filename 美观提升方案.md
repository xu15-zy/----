# 敦煌纹样网站 · 桌面端美观度提升方案（图形化 / 装饰化路线）

> 范围：仅桌面端视觉美化，不做移动端。
> 方向确认：**不追求写实照片**，走「图形化文化装饰」路线——用 SVG 纹样、藻井、印章、联珠边框等母语级视觉符号出彩，强调好看 + 敦煌文化特征。
> 配色与字体已统一，本方案只做「叠加美化」，不动基础系统。

---

## 一、最高性价比（签名级文化图形，先做）

### 1. 宣纸纹理 + 阴影系统（材料感，非写实）
给全站一层极淡噪点 + 真实阴影，立刻有「纸本/壁画」实物感。

body::before {

### 2. 藻井背景（Hero 签名主视觉）⭐
藻井 = 莫高窟窟顶的「叠套方井 + 中心圆」图案，是敦煌最 iconic 的图形。用它做 Hero 背景水印或主视觉，比照片更有「文化辨识度」。

`components/Caisson.tsx`（可复用于 Hero 背景 / 区块分隔）：
```tsx
export default function Caisson({ className = "", stroke = "#c9a227", opacity = 0.08 }: { className?: string; stroke?: string; opacity?: number }) {
  const pearls = Array.from({ length: 24 }, (_, i) => {
    const a = (i / 24) * Math.PI * 2;
    return <circle key={i} cx={200 + 92 * Math.cos(a)} cy={200 + 92 * Math.sin(a)} r={3} fill={stroke} />;
  });
  return (
    <svg className={className} viewBox="0 0 400 400" fill="none" stroke={stroke} style={{ opacity }}>
      <rect x="40" y="40" width="320" height="320" strokeWidth={2} />
      <rect x="80" y="80" width="240" height="240" strokeWidth={2} transform="rotate(45 200 200)" />
      <rect x="120" y="120" width="160" height="160" strokeWidth={2} />
      <circle cx="200" cy="200" r="62" strokeWidth={2} />
      <circle cx="200" cy="200" r="30" strokeWidth={2} />
      {pearls}
    </svg>
  );
}
```
Hero 用法：`<div className="absolute inset-0 -z-10 grid place-items-center"><Caisson className="h-[120%] w-auto" /></div>` —— 居中巨幅藻井，文字浮其上。

### 3. 联珠纹 + 卷草角饰边框（装饰框系统）
敦煌边饰的灵魂是「联珠（圆点串）」与「卷草」。做成可复用类，套在 3D 容器、CTA 卡、LayerRecombine 容器上。

`app/globals.css`：
```css
/* 联珠纹分割带 */
.divider-pearl { display: flex; align-items: center; gap: 14px; }
.divider-pearl > span { height: 8px; width: 8px; transform: rotate(45deg); background: #b23a2e; }
.divider-pearl::before, .divider-pearl::after {
  content: ""; flex: 1; height: 8px; opacity: .5;
  background-image: radial-gradient(circle, #c9a227 1.6px, transparent 2px);
  background-size: 16px 8px; background-repeat: repeat-x;
}
/* 卷草角饰：贴容器四角 */
.frame-dh { position: relative; }
.frame-dh::before, .frame-dh::after {
  content: ""; position: absolute; width: 22px; height: 22px; border: 1.5px solid #c9a227; opacity: .7;
}
.frame-dh::before { top: 10px; left: 10px; border-right: 0; border-bottom: 0; }
.frame-dh::after  { bottom: 10px; right: 10px; border-left: 0; border-top: 0; }
```

### 4. 印章式标签（朱砂方印，文化签名）⭐
「印章」是中国视觉里辨识度极高的文化符号，做起来极简。用于页面序号、栏目落款、Hero 点缀。

`components/Seal.tsx`：
```tsx
export default function Seal({ text, size = 48 }: { text: string; size?: number }) {
  return (
    <span
      className="inline-flex items-center justify-center rounded-md bg-zhusha font-serif text-mibai-light shadow-dh-sm [writing-mode:vertical-rl]"
      style={{ width: size, height: size, fontSize: size * 0.34, letterSpacing: "0.1em" }}
    >
      {text}
    </span>
  );
}
```
示例：栏目标题左侧放 `<Seal text="敦煌" />`，或详情页用 `<Seal text="忍冬" />` 作落款。两字竖排方印，气质立刻到位。

## 二、中等投入（结构节奏 + 图形化表达）

### 5. 区块交替底色 + 藻井分隔带
相邻 section 交替 `米白-light / 米白-deep`，长滚动不疲劳；大区块之间用 `.divider-pearl` 与 `.divider-dh` 交替。

```css
.band { background: #fbf6ea; }
.band-deep { background: #e8dcc0; }
```

### 6. Hero 左右布局：左文 + 右放大线描
把现有 `PatternAnimation` 的 motif 放大成「主视觉」放右侧，左侧文字，比纯居中更有设计感：
- 右侧用 `Caisson` 巨幅藻井衬底 + 中心放一个放大的「莲花绽放 / 三兔共耳」线描（`dh-bloom` / `dh-orbit` 动画）。

### 7. 竖排中文侧标
Hero 两侧或详情页 Hero 左缘放竖排小标，强化「国风」：
```css
.vtitle { writing-mode: vertical-rl; letter-spacing: .35em; color: #9c5b3b; font-family: "Noto Serif SC", serif; }
```
文字如「敦煌 · 六大核心纹样」「千年营建 · 文明交融」。

### 8. 语义化字号体系
`tailwind.config.ts` 扩展（保留默认）：
```ts
fontSize: {
  display:  ["3.5rem",  { lineHeight: "1.05", letterSpacing: "-0.01em" }],
  title:    ["2.25rem", { lineHeight: "1.15" }],
  heading:  ["1.5rem",  { lineHeight: "1.3" }],
  body:     ["1rem",    { lineHeight: "1.75" }],
  small:    ["0.875rem",{ lineHeight: "1.6" }],
  caption:  ["0.75rem", { lineHeight: "1.4" }],
},
```
Hero 用 `text-display`、Section 标题 `text-title`、卡片标题 `text-heading`，层级更稳。
