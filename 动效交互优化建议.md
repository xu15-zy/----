# 敦煌纹样网站 · 动效交互优化建议

> 桌面端优先，图形化装饰方向（不写实）。所有建议均贴现有代码（`globals.css` / `Reveal.tsx` / `PatternAnimation.tsx` / `LayerRecombine.tsx` / `ModelViewer.tsx`）。
> 优先级：P0 体验与稳定性底线 · P1 核心交互升级 · P2 微交互打磨。

---

## 0. 现状盘点

| 动效 | 位置 | 实现 | 状态 |
|------|------|------|------|
| 滚动揭示 | `Reveal.tsx` + `.reveal` | IO + translateY(28px)+opacity, 0.7s | 一次性，OK |
| 卡片 hover | `.card-dh` | -translate-y-1 + shadow-md, 0.3s | 偏弱 |
| 按钮 hover | `.btn-dh` | 仅变色 | 无位移/光泽 |
| 差异动画 | `PatternAnimation.tsx` | 6 套 SVG 关键帧（grow/bloom/drift/ripple/dot/orbit） | 标注"待替换 MP4" |
| 图层重组 | `LayerRecombine.tsx` | 三层圆 opacity 切换, 0.4s | 无动感/无联动 |
| 3D 查看器 | `ModelViewer.tsx` | OrbitControls 自动旋转 + 拖拽缩放 | Suspense=null 闪白；Environment CDN 依赖 |
| 全局 | `globals.css` | `scroll-behavior: smooth` | reduced-motion 未关 |

**三个根因**：① 缓动/时长没有统一 token；② `prefers-reduced-motion` 只覆盖 `.reveal`，那批无限循环 SVG 动画照转；③ 微交互（hover 反馈、路由过渡、进场、cursor）几乎空白。

## P0 · 体验与稳定性底线

### P0-1 全局 reduced-motion 兜底（必做，无障碍）

当前只关了 `.reveal`，但 `dh-grow / dh-bloom / dh-drift / dh-ripple / dh-dot / dh-orbit` 仍在无限循环，对前庭敏感用户会诱发眩晕。

```css
/* globals.css — 替换现有那条只管 .reveal 的 media query */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.001ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.001ms !important;
    scroll-behavior: auto !important;
  }
  /* 入场动画的终态必须可见（dh-bloom 用 both 会停在初始隐藏态） */
  .dh-bloom,
  .dh-bloom * { transform: none !important; opacity: 1 !important; }
}
```

### P0-2 JS 失效兜底（避免内容永久隐藏）

`.reveal` 默认 `opacity:0`，禁用 JS 时内容永远不可见。改为「默认可见，有 JS 才隐藏」：

```css
/* globals.css */
.reveal { opacity: 1; transform: none; }            /* 默认态：可见 */
html.js .reveal {                                    /* 仅 JS 开启时隐藏待播 */
  opacity: 0;
  transform: translateY(28px);
  transition: opacity 0.7s var(--ease-dh), transform 0.7s var(--ease-dh);
  will-change: opacity, transform;
}
html.js .reveal.in { opacity: 1; transform: none; }
```
`Reveal.tsx` 顶部加一行 `useEffect(() => { document.documentElement.classList.add('js') }, [])`。

### P0-3 3D 加载不再闪白 + 去掉 CDN 光照依赖

`Suspense fallback={null}` 加载期白屏；`Environment preset="sunset"` 从 drei CDN 拉 HDR，离线/慢网直接黑。

```tsx
// ModelViewer.tsx — Suspense 外置 DOM 加载层 + loading 状态
const [loading, setLoading] = useState(true);
// Model 内 useEffect 末尾：setLoading(false)
// 容器：
<div className="relative h-full w-full">
  {loading && (
    <div className="absolute inset-0 z-10 flex flex-col items-center justify-center gap-3 bg-mibai-deep/40">
      <div className="h-14 w-14 animate-slow-spin rounded-full border-2 border-dashed" style={{ borderColor: accent }} />
      <p className="text-xs text-mo/50">模型渲染中…</p>
    </div>
  )}
  <Canvas ...>
    {/* 删除 <Environment preset="sunset" />，改用本地灯光 rig */}
    <ambientLight intensity={1.1} />
    <directionalLight position={[4,5,3]} intensity={1.5} />
    <directionalLight position={[-3,2,4]} intensity={0.7} />
    <pointLight position={[0,2,0]} intensity={0.5} color={accent} />
    <Suspense fallback={null}><Model url={src} accent={accent} onReady={() => setLoading(false)} /></Suspense>
  </Canvas>
</div>
```

## P1 · 核心交互升级

### P1-1 统一缓动与时长 token（所有动效的地基）

```css
:root {
  --ease-dh: cubic-bezier(0.16, 1, 0.3, 1);   /* 优雅减速，敦煌「舒展」感 */
  --dur-fast: 200ms;
  --dur-base: 320ms;
  --dur-slow: 600ms;
}
```
- 卡片/按钮 hover 改用 `transition: ... var(--dur-base) var(--ease-dh)`
- `.reveal` 改用 `var(--ease-dh)`
- 所有 `dh-*` 关键帧的 `ease-out` 统一为 `var(--ease-dh)`（在 class 上写 `animation: dh-grow 3s var(--ease-dh) infinite alternate`）

### P1-2 滚动揭示增强（错峰 + 多形态）

现状只有「上移」，建议支持方向，并支持容器级联 stagger：

```tsx
// Reveal.tsx 增加 variant
type Variant = "up" | "scale" | "left" | "blur";
const MAP = {
  up:   "translateY(28px)",
  scale: "scale(0.96)",
  left:  "translateX(-24px)",
  blur:  "translateY(20px) blur(6px)",
};
// className 拼 `reveal reveal-${variant}`；CSS 里对每种写 transform 初态
```
区块标题用 `up`、纹样卡网格用 `scale`、侧栏用 `left`，区分阅读节奏。长列表 stagger 用 `delay = i * 80`。

### P1-3 3D 进场 + 拖拽反馈 + 视角限制

```tsx
// 进场：模型首次出现从缩小淡入
<Canvas onCreated={({ camera }) => { camera.position.set(0, 1.2, 5.2); }}>
  <group ref={wrap} style={{ /* 用 useFrame 做 0.8s 缓动入场 */ }}>

// 拖拽 cursor 反馈（CSS）
canvas { cursor: grab; }
canvas:active { cursor: grabbing; }

// OrbitControls 限制，避免转到模型下方穿模
<OrbitControls
  autoRotate={auto}
  enablePan={false}
  enableZoom
  minDistance={2.4}
  maxDistance={8}
  minPolarAngle={Math.PI / 6}
  maxPolarAngle={Math.PI - Math.PI / 6}
  autoRotateSpeed={0.8}   /* 放慢，减少眩晕感 */
/>
```
首入建议**关闭自动旋转**，用户点「自动旋转」再转（自动转对想细看的人是干扰）。

### P1-4 图层重组做「动感 + 联动」

现状只切 opacity，切换不直观。建议：切换时该层做一次轻微「脉冲」（scale 1→1.06→1），并 **hover 按钮时高亮对应层**，建立映射：

```tsx
// LayerRecombine.tsx
const [hover, setHover] = useState<keyof typeof show | null>(null);
const ring = (k) => ({
  opacity: show[k] ? 1 : 0.12,
  transform: hover === k ? "scale(1.05)" : "scale(1)",
  transition: "opacity .4s var(--ease-dh), transform .4s var(--ease-dh)",
});
// 按钮加 aria-pressed={show[row.key]} 与 onMouseEnter/Leave 设 hover
```
三层同心时，给每层不同的 `strokeDasharray` 旋转角，切换更有「重组」语义。

### P1-5 差异动画正式化（贴合图形化方向，丢掉 MP4 计划）

用户要「图形化不写实」，那批 SVG 动效不该等 MP4 替换——直接做成**正式纹样微动效库**：

- 去掉容器里「差异动画占位（待替换 MP4）」字样；
- 节奏统一为「缓慢、有呼吸感、可循环但克制」：`dh-ripple / dh-dot / dh-orbit` 周期收到 4–6s，`dh-drift` 幅度从 ±10px 提到 ±16px 更可见；
- 作为 section 背景或图标点缀复用，而非只挤在 224px 小框里。

## P2 · 微交互打磨

### P2-1 按钮 hover 升级

```css
.btn-dh {
  position: relative; overflow: hidden;
  transition: background var(--dur-base) var(--ease-dh), transform var(--dur-fast) var(--ease-dh);
}
.btn-dh:hover { transform: translateY(-1px); }
.btn-dh:active { transform: translateY(0); }
.btn-dh::after { /* 金色扫光 */
  content: ""; position: absolute; inset: 0;
  background: linear-gradient(120deg, transparent 30%, rgba(255,255,255,.35) 50%, transparent 70%);
  transform: translateX(-120%); transition: transform .6s var(--ease-dh);
}
.btn-dh:hover::after { transform: translateX(120%); }
```

### P2-2 链接金色下划线扫光（替代单纯变色）

```css
.link-dh {
  background-image: linear-gradient(var(--jin), var(--jin));
  background-size: 0% 1.5px; background-repeat: no-repeat; background-position: 0 100%;
  transition: background-size var(--dur-base) var(--ease-dh);
}
.link-dh:hover { background-size: 100% 1.5px; }
```

### P2-3 卡片 hover 金边 + 中阴影

```css
.card-dh:hover {
  border-color: rgb(201 162 39 / .55);          /* jin */
  box-shadow: 0 14px 30px -12px rgb(43 38 32 / .28);
  transform: translateY(-4px);
}
```

### P2-4 Nav 链接活动指示

```css
.nav-link { position: relative; }
.nav-link::after {
  content: ""; position: absolute; left: 0; bottom: -4px; height: 2px; width: 0;
  background: var(--jin); transition: width var(--dur-base) var(--ease-dh);
}
.nav-link:hover::after, .nav-link[data-active]::after { width: 100%; }
```

### P2-5 路由切换过渡（Next.js `template.tsx`）

```tsx
// app/template.tsx
"use client";
import { motion } from "framer-motion";  // 或纯 CSS .page-enter
export default function Template({ children }: { children: React.ReactNode }) {
  return <div className="page-enter">{children}</div>;
}
```
```css
.page-enter { animation: pageIn var(--dur-slow) var(--ease-dh) both; }
@keyframes pageIn { from { opacity: 0; transform: translateY(12px); } to { opacity: 1; transform: none; } }
```

### P2-6 Hero 进场编排

标题逐行揭示（用 `.reveal` 多实例 + stagger）；右侧藻井 `caisson-hero.png` 做 `animation: dh-orbit 90s linear infinite`（极慢自转，呼吸感）；印章落款淡入延迟出现。

**一句话**：先把「减动偏好兜底 + 3D 不闪白 + 缓动统一」三件做了，整个站点的动效就从「零散拼凑」变成「有章法的敦煌节奏」；再补图层联动和微交互，质感就能追上视觉。需要我直接动手实现其中某批，说一声就开干。
